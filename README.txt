Aegis Flutter + Firebase Prototype

What is included:
- Flutter entry point
- Firebase initialization with placeholder firebase_options.dart
- Login screen
- Overview dashboard
- Deception Ops screen
- AI Copilot screen
- Reports screen
- Firestore rules
- Firebase hosting config

Run steps:
1. flutter create aegis_flutter_firebase
2. Replace lib/, pubspec.yaml, firebase.json, and firestore.rules with these files
3. Run flutterfire configure and replace lib/firebase_options.dart
4. flutter pub get
5. flutter run -d chrome
